**Day 28 - Exploratory Data Analysis (EDA) with Titanic Dataset**

### Topics Covered

* Loaded and explored the Titanic dataset using Pandas.
* Performed basic data inspection using:

  * `head()`
  * `tail()`
  * `info()`
  * `describe()`
  * `dtypes`
* Analyzed categorical data using `value_counts()`.
* Used normalized frequencies (`normalize=True`) to calculate proportions.
* Counted male and female passengers and compared survival proportions.
* Found unique values using `nunique()`.
* Applied `groupby()` with `mean()` to summarize numeric data.
* Calculated covariance and correlation between **Age** and **Fare**.
* Created a random sample from the dataset using `sample()` and compared its statistics.

### Practice Analysis

* Analyzed survival based on:

  * Gender
  * Passenger Class
  * Age Group (created using `pd.cut()`)
  * Fare Group (created using `pd.cut()`)
* Drew conclusions about which passengers had higher survival chances.

### Embarkation Port Analysis

* Counted passengers from each embarkation port.
* Calculated survival rate for each port using `groupby()`.
* Identified the port with the highest survival rate.

### Key Concepts Learned

* Exploratory Data Analysis (EDA)
* Data inspection and summarization
* Frequency and proportion analysis
* GroupBy operations
* Normalization
* Correlation and covariance
* Random sampling
* Feature engineering using `pd.cut()`
* Interpreting insights from data
